How to Use the API

Call the endpoint below while a SynthSpeak session is active to get live candidate metrics.

POST http://your-server:8000/api/analyze
curl -X POST http://localhost:8000/api/analyze \
  -H "Authorization: Bearer ss_your_api_key"
{
  "metrics": {
    "similarity": 0.81,
    "is_on_topic": true,
    "eye_contact_percentage": 74.3,
    "posture_score": 88.0,
    "filler_word_count": 3,
    "total_word_count": 120,
    "transcription": "Today I'll show you...",
    "retrieved_at": "2026-03-10T18:00:00"
  }
}
GET /api/interview-report/{session_id}
Authorization: Bearer ss_your_api_key
